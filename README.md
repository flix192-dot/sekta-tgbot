# 🤖 Sekta Bot

Telegram бот для клуба Секта — уведомления, события, привычки, голосовые сообщения.

## Файлы проекта

```
bot.py            ← основной код бота
requirements.txt  ← зависимости Python
railway.toml      ← настройки деплоя
README.md         ← эта инструкция
```

> ⚠️ На GitHub нужно загружать **все 4 файла**, иначе Railway не установит зависимости и Gemini/Whisper работать не будут.

## Переменные окружения (Railway)

| Переменная | Обязательная | Описание |
|---|---|---|
| `BOT_TOKEN` | ✅ | Токен от @BotFather |
| `FIREBASE_URL` | ✅ | URL базы Firebase |
| `TG_ARTEM` | ✅ | Telegram ID Артёма (число) |
| `TG_BOGDAN` | ✅ | Telegram ID Богдана (число) |
| `FIREBASE_CREDS` | ✅ | JSON service account одной строкой |
| `GEMINI_API_KEY` | ⚡ | Ключ Google AI Studio (умный парсинг + чат) |
| `GROQ_API_KEY` | 🎤 | Ключ Groq (транскрипция голосовых через Whisper) |

Без `GEMINI_API_KEY` и `GROQ_API_KEY` бот работает, но без AI-функций.

## Деплой на Railway

1. Создай репозиторий на GitHub
2. Загрузи **все 4 файла** (bot.py, requirements.txt, railway.toml, README.md)
3. Railway → New Project → Deploy from GitHub repo
4. Добавь все переменные окружения
5. Railway сам запустит и перезапустит бота

## Как получить ключи

**GEMINI_API_KEY** — бесплатно:
→ https://aistudio.google.com → Get API Key

**GROQ_API_KEY** — бесплатно:
→ https://console.groq.com → API Keys → Create

## Команды бота

| Команда | Действие |
|---|---|
| `/start` | Приветствие и статус |
| `/help` | Справка |
| `/today` | События на сегодня |
| `/list` | Все события |
| `/remind` | Настроить напоминания |

## Примеры

**Текстом:**
```
завтра в 14 тренировка
поставь встречу послезавтра в 19:00 на 1.5 часа
03.07 в 10 подготовка к НМТ
удали план тренировка завтра
```

**Привычки:**
```
потренировался  → попросит фото из зала
прочитал 30 страниц  → галочка сразу
встал в 7:30  → галочка ранний подъём
```
